# Project Scope and Objectives

# Project Title

AI-Powered Job Recruitment & Candidate Management System (AIRCMS)

---

# Project Scope

## Overview

The AI-Powered Job Recruitment & Candidate Management System (AIRCMS) is a Salesforce-based recruitment platform developed using Salesforce Admin tools, Flow Builder, Approval Processes, Lightning Experience, and Agentforce AI. The project is designed to automate and streamline the complete recruitment lifecycle without using custom Apex development.

The application supports the following recruitment process:

Candidate → Screening → Interview → Offer → Hiring

---

# In Scope

The project includes the following features:

## Candidate Management
- Create and manage candidate profiles
- Store candidate contact details
- Track candidate skills and experience
- Maintain application history

## Job Management
- Create job openings
- Manage departments
- Track vacancies
- Assign recruiters

## Recruitment Process
- Candidate application tracking
- Interview scheduling
- Interview feedback management
- Offer generation
- Hiring status tracking

## Automation
- Record-triggered Flows
- Email notifications
- Recruiter reminders
- Automatic status updates

## AI Features
- AI-generated candidate summaries
- Candidate scoring
- Hiring recommendations
- Risk analysis

## Reports & Dashboards
- Recruitment dashboards
- Candidate reports
- Recruiter performance reports
- Hiring analytics

## Security
- Role-based access
- Profiles
- Permission Sets
- Approval Processes

---

# Out of Scope

The following features are not included in this project:

- Custom Apex Classes
- Apex Triggers
- External HR integrations
- Payroll Management
- Employee Attendance Management
- Video Interview Platform
- Mobile Application
- Third-party Recruitment Portals

---

# Project Objectives

The objectives of AIRCMS are:

- Implement Salesforce CRM fundamentals for recruitment.
- Build a scalable recruitment data model.
- Centralize candidate and job information.
- Automate recruitment workflows using Salesforce Flows.
- Implement AI-powered candidate evaluation using Agentforce.
- Apply role-based security and governance.
- Support approval-driven hiring decisions.
- Improve recruiter productivity.
- Reduce manual recruitment activities.
- Generate real-time recruitment reports and dashboards.
