# Milestone 3: Tabs Creation

## Overview

In this milestone, custom tabs are created for the **Job Opening** and **Candidate Application** objects. Tabs provide users with an easy way to access custom objects directly from the Salesforce navigation menu, improving usability and navigation within the Recruitment Management System.

---

## Objective

The objectives of this milestone are to:

* Create custom tabs for the recruitment objects.
* Improve accessibility to custom records.
* Configure tab visibility for user profiles.
* Prepare the objects for integration into Salesforce applications.

---

## Activity 1: Creating the Job Opening Tab

A custom tab is created for the **Job Opening** object to allow users to manage job openings directly from the Salesforce interface.

### Steps

1. Open **Salesforce Setup**.
2. In the **Quick Find** box, search for **Tabs**.
3. Select **Tabs**.
4. Under **Custom Object Tabs**, click **New**.
5. Choose **Job Opening** as the object.
6. Select a preferred **Tab Style**.
7. Click **Next**.
8. Keep the default profile settings.
9. Click **Next**.
10. On the **Add to Custom Apps** page, uncheck **Include Tab**.
11. Ensure **Append tab to users' existing personal customizations** is selected.
12. Click **Save**.

---

## Activity 2: Creating the Candidate Application Tab

A custom tab is created for the **Candidate Application** object to provide direct access to candidate records.

### Steps

1. Open **Salesforce Setup**.
2. Search for **Tabs** in the **Quick Find** box.
3. Select **Tabs**.
4. Under **Custom Object Tabs**, click **New**.
5. Choose **Candidate Application** as the object.
6. Select a preferred **Tab Style**.
7. Click **Next**.
8. Keep the default profile settings.
9. Click **Next**.
10. Uncheck **Include Tab** under **Add to Custom Apps**.
11. Ensure **Append tab to users' existing personal customizations** is enabled.
12. Click **Save**.

---

## Expected Outcome

After completing this milestone:

* A custom tab for **Job Opening** is successfully created.
* A custom tab for **Candidate Application** is successfully created.
* Both objects become easily accessible through Salesforce navigation.
* Default profile visibility is configured.
* The tabs are available for future use in custom applications.
